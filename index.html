Viking GPS Website

Viking GPS Desktop Application.

This is currently a replication website for Viking which is primarily hosted on SourceForge.

Viking is a free/open source program to manage GPS data (including GPX and KML files). You can import and plot tracks, routes and waypoints, show OpenStreetMaps OSM, Bing Aerial and other maps, generate Mapnik maps, geotag images, make new tracks, routes and waypoints, see real-time GPS position, etc...

It is written mostly in C with the GTK+ 2 toolkit and some C++.

Issues and Pull requests are welcomed via Viking on GitHub, as well as the standard patches and bug/features hosted on SourceForge.
